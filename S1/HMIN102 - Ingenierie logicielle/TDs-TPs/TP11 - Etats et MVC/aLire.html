Pour lire le corrigé de Calculette avec State version Standard : cliquer sur le lien EnseigCalcState.st

Pour lire le corrigé de Calculette version non standard, présentée en cours utilisant adoptInstance : cliquer sur le lien Calc.st

Pour exécuter ce corrigé :

Enregistrer le fichier EnseigCalcState.st quelque part,
Ouvrir PharoSmalltalk,
Dans le menu principal, choisir "TOOLS>FILE-BROWSER",
Trouver le fichier,
Cliquer sur "Filein"

Le contenu du fichier est alors chargé, vous pouvez retrouver le code dans le "browser" avec "find class", chercher "Calc".
